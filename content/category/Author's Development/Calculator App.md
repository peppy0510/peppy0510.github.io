#### Description

* 하이브리드 앱
* 계산기
* 수식과 결과 히스토리 기능
* JavaScript Math 함수 사용 가능
* 자릿수 구분기호 표기 기능 지원
* 자동 반올림, 올림, 내림 기능 지원

---

#### Development Stack

* IONIC 3, TypeScript, JavaScript, HTML, SCSS
